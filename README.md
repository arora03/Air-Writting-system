# ✨ Air Writing System

A real-time gesture-based air writing system using computer vision.

## 🚀 Features
- Hand tracking using MediaPipe
- Gesture-based control system
- Sticky mode + confirmation logic
- Smooth air drawing engine
- Canvas capture for ML (in progress)

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## 🎮 Controls
- ☝️ Index → Write
- ✌️ Two fingers → Erase
- ✊ Fist → Pause
- 👍 Thumb → Predict (coming soon)

## 📌 Future Work
- Character recognition (CNN)
- Word formation
- UI improvements
